## Why use npmx.dev?

npmx.dev focuses on:

- **Speed** - Fast page loads with no layout shift
- **Developer experience** - Dark mode, translations, accessibility, keyboard navigation, and shareable URLs
- **Insights** - Module format badges, install size, version diffs, package comparison, and dependency health signals
- **Compatibility** - Works with existing npm URLs
